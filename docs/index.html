Vintage IPA installer for jailbroken iOS 6+. Browse 157,000 archived apps (2008–2014) and install with one tap.

Add this source in Cydia / Sileo

https://AdrienRL1.github.io/adrienrl/

Steps:

  1. Open Cydia (or Sileo).
  2. Sources → Edit → Add.
  3. Paste the URL above. Tap Add Source.
  4. Find AppDrop under the new source. Tap Install.

Or grab the .ipa directly

If you don't want a Cydia source, the .ipa is on the GitHub Releases page:

Releases →

Install via Filza or ipainstaller AppDrop.ipa.

Compatibility

  • iOS 6.0 – 9.3.6 (armv7 / 32-bit)
  • iPhone 4S, iPad 2 / 3 / 4 / iPad mini, iPod touch 5
  • Requires a jailbreak + IPA Installer Console (autopear) or AppInst
